# FePy

Finite Element Python Package

- fepy
  - fem: 有限元流程框架
  - mesh: 网格生成框架
    - 一维网格
    - 二维网格
      - 三角单元
      - 矩形单元
  - boundary: 边界处理方法
    - Dirichlet 边界
    - Neumann 边界
    - Robbin 边界
  - error: 误差估计方法
    - 先验误差估计
    - 后验误差估计
  - adaptive: 自适应流程框架
  - store: 数据存储规范
  - vision: 可视化方法
